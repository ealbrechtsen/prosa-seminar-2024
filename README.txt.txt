DiceWar Project
================

Author: Emil Malik Albrechtsen

### Description:
This project is a simple dice game implemented in C#. 
It follows the requirements outlined in the assignment, 
including the creation of classes such as 
`Dice`, `WarDice`, `WarDiceCup`, and `Game`.  

The project includes:
- `DiceWar.csproj` (project file)
- Necessary `.cs` files 
- A `.sln` (solution file)

### Instructions:
1. Extract the ZIP file to a folder.
2. Open a terminal/command prompt.
3. Navigate to the project folder.
4. Run the following command:

dotnet build
dotnet run
